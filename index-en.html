How to Use This Mirror Site?

If you want to use the acceleration service, simply replace cdn.jsdelivr.net with jsdelivrcn.netlify.app in the jsDelivr link. For example:

https://cdn.jsdelivr.net/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

After replacement, it becomes:

https://jsdelivrcn.netlify.app/npm/jquery@4.0.0-beta.2/dist/jquery.min.js

To access the project folder, use the following format:

https://jsdelivrcn.netlify.app/npm/package-name@version/

Example:

https://jsdelivrcn.netlify.app/npm/jquery@4.0.0-beta.2/


To ensure the stability and compliance of the service, please adhere to the following regulations when using this mirror site:

  • Do not accelerate or distribute content that violates the laws and regulations of Mainland China, including but not limited to pornography, violence, and politically sensitive information.
  • The mirror site provides stable acceleration services, but does not guarantee long-term availability.
  • If you need to clear the cache, please refer to the jsDelivr official guide.
